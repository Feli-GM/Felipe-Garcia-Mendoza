For this project I installed the Cursor IDE program and the GitHub program

First I downloaded Cursor IDE and followed the instructions to install it. I created an account in order to use it. After that, I searched for the Claude Code add-on and installed the one made by Anthropic. I did the same for the Codex add-on from OpenAI. After completing these steps, I went to GitHub and created my account. On the website, I created the public repository and added a README.md file from settings. I then downloaded the GitHub program for Windows and used the option to open the repository directly in Cursor IDE.

The first issue I ran into is that I was not sure how to open my repository in Cursor IDE. I restarted Cursor IDE and found the option to clone the repository but for some reason it was not working. I asked ChatGPT what the reason might be and that is when it suggested to install the GitHub program. I opened the repository in the GitHub program and there I found the option to open it in Cursor IDE.
